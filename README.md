# Clean Architecture .NET Template

This repository provides a reusable Clean Architecture template for .NET Web APIs.

## Structure
- Domain: Core business entities and interfaces
- Application: Business logic and use cases
- Infrastructure: External systems (DB, AD, Email)
- API: HTTP entry point

## Principles
- Loosely coupled
- Dependency inversion
- Testable by design
- Framework-independent domain

## How to Use
1. Clone repository
2. Rename Template.* projects
3. Start with Domain layer
